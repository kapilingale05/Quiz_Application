# 🎯 Python Quiz Application (GUI Version)
A Sudden Death Quiz Application built using Python and Tkinter GUI.

## 🚀 Features
- Randomized Questions
- Randomized Options
- Sudden Death Mode (Quiz ends on first wrong answer)
- Total Time Tracking
- JSON-based Question Storage
- Simple GUI using Tkinter

## 🛠 Technologies Used
- Python
- Tkinter (GUI)
- JSON
- OOP Concepts
